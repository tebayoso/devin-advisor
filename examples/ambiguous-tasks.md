# Example Ambiguous Tasks

Use these to try the devin-scope pipeline. Each is intentionally underspecified so the decomposition
and adversarial review have something to work with.

1. "Implement rate limiting for the public API."
2. "Add SSO login to the dashboard."
3. "Migrate the service from REST to GraphQL."
4. "Make the app faster."
5. "Add offline support to the mobile app."
6. "Improve test coverage for the payments module."
7. "Set up observability for the checkout flow."
